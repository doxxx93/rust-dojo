# 3.1 Variables and Mutability

> 시작: 2026-02-04

## 요약

변수와 가변성


## 주요 개념

- immutable variable와 constants는 이름과 값이 서로 묶여있다.(bound)
- Constants에는 mut 을 사용하지 못한다. `let`이 아닌 `const`로 선언하며 value의 타입이 명시되어야 한다.
- Constant는 런타임에서 계산된 값이 아닌 오직 상수 표현식(constant expression)으로만 초기화될 수 있다.
